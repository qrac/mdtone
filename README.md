# MDTONE

## Demo

- [Demo Page][link-demo]

## About

Material Design ColorをSassで使いやすいよう変数化しました。

## Use

### 色をWebデザインで使う場合

[Demo Page][link-demo]に16進数の色コードを自動出力していますので、コピーしてお使いください。

### SCSSの変数として利用する場合

`/src/scss/variable/_mdtone.scss`をお使いください。
